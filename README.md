# AWS EC2 Website Deployment
## 🌐 Live Demo
http://13.42.12.227
## Project Overview
This project demonstrates how I deployed a static website on AWS EC2 using Amazon Linux and Apache HTTP Server.

## Technologies Used
- AWS EC2
- Amazon Linux
- Apache HTTP Server
- SSH
- SCP
- Windows PowerShell

## What I Did
- Launched an EC2 instance
- Configured security groups for SSH and HTTP
- Connected to the server using SSH and a PEM key
- Installed and enabled Apache HTTP Server
- Uploaded website files to the EC2 instance
- Deployed the website to `/var/www/html`
- Verified the live site using the EC2 public IP

## Skills Demonstrated
- AWS EC2 provisioning
- Linux server administration
- Apache web server configuration
- SSH remote access
- Static website deployment
